【Java】Class初始化过程是什么
一、加载阶段
1.通过类的全限定名后去类的二进制字节流
2.将字节流的静态存储结构转化为方法区的运行时数据结构
3.在内存中生成 Class对象，作为方法去这个类的数据访问接口
总的来说类型数据放在方法区，java堆中实例化Class类的对象
二、验证阶段
验证:格式验证、元数据验证、字节码验证、符号引用验证。
三、准备阶段
为类的静态变量分配内存，初始化默认值
四、解析阶段
把类或接口、字段类方法、接口方法、方法类型、方法句柄、限定符
符号应用转换为直接执行目标的指针或句柄
将编译后的虚拟地址和实际运行地址关联起来
五、初始化
jvm对类进行初始化，主要是对类变量初始化
