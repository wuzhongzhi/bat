介绍线程的生命周期和状态？
Java线程有以下五种状态，对应其生命周期中的不同阶段：
新建（New）状态：当线程对象被创建时，线程处于新建状态，此时还没有启动线程。
运行（Runnable）状态：当调用线程对象的start()方法后，线程进入运行状态。
阻塞（Blocked）状态：线程处于阻塞状态，、
是因为它等待某个操作完成（如等待输入、输出、等待获取锁等），在等待期间它不会占用CPU资源。
等待（Waiting）状态：线程处于等待状态，
是因为它等待其他线程通知唤醒它，进入等待状态的线程会释放它所持有的锁，以便其他线程能够获得该锁并执行。
终止（Terminated）状态：当线程执行完run()方法或者发生了未捕获的异常时，线程就会进入终止状态。
